Cultural Resource Survey Requests

Embeded below is a list of the cases for which cultural resource surveys have been requested. They are grouped by parish, which will assist in making efficient travel plans for site work, and also reveals that numerous "cases" are simply multiple construction phases of the same larger development parcel and for the same applicant. Note the right-most field contains graphics. These are air photos, site plans, or similar documents which define the location and extent of the parcels.

Note that the priority for conducting these surveys is "oldest Phase I ESA date first", not in alphabetic order by parish name.
