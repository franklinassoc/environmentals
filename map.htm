Status Tracking Map

This Google map strives to display cases with location pins color coded to case status. Locations are based on address geocodes, and only one address is chosen for scattered-site housing cases, so pin locations often are not exact. Cases having identical property addresses (though separate phases) are displayed as a single map pin.
